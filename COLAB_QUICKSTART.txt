STEP 1: Upload Your Code to GitHub (One-time)
----------------------------------------------
1. Open terminal in your project folder
2. Run these commands:
   git add .
   git commit -m "Ready for Colab"
   git push
3. Note your GitHub username for next step


STEP 2: Open Colab Notebook
----------------------------
1. Go to: https://colab.research.google.com/
2. Click: File ‚Üí Upload Notebook
3. Select: colab_complete_pipeline.ipynb (from your project folder)


STEP 3: Setup GPU
-----------------
1. In Colab, click: Runtime ‚Üí Change runtime type
2. Select: T4 GPU
3. Click: Save


STEP 4: Update GitHub Username
-------------------------------
In Cell 3, change this line:
  !git clone https://github.com/YOUR_USERNAME/Axion-Sat.git
To your actual GitHub username


STEP 5: Run Everything
----------------------
1. Click: Runtime ‚Üí Run all
2. When asked, click "Connect to Google Drive" and authorize
3. Sit back and relax! Everything happens automatically:
   ‚úì Downloads BigEarthNet data
   ‚úì Converts to paired tiles
   ‚úì Runs Stage 1 precompute
   ‚úì Saves to Google Drive
4. Total time: ~20-25 hours


STEP 6: Download Results
-------------------------
1. Go to Google Drive: Axion-Sat-Outputs/stage1_precompute
2. Download all files (or sync with Google Drive Desktop)
3. Save to: D:\axion_stage1_precompute\
